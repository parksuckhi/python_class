  ### for 문 이용
   - 리스트[1]부터 출력
  ```
  리스트 = ["가", "나", "다", "라"]
  for 변수 in 리스트[1:]:
  print(변수)
  ```
  
  - 2자리를 증가해서 건너뜀
  ```
  리스트 = ["가","나","다","라"]
  for 변수 in 리스트[::2]:
  print(변수) 
  ```
  
  - 리스트를 거꾸로 나오게 출력
  ```
  리스트 = ["가","나","다","라"]
  for 변수 in 리스트[::-1]:
  print(변수)
  ```
  ### if 문 이용
  - 변수가 음수만 나오게 한다. if문을 이용해서 
  ```
  if 변수 < 0:
  ``` 
  - 리스트 중 3의 배수만 나오게 함(나머지 이용)
  ```
  if 변수 % 3 == 0:
  ```
  -20보다 작고 3의 배수라는 조건 둘다 만족하는 수 출력
  ```

  ```
