  ### for 문 이용
   - 리스트[1]부터 출력
  ```
  리스트 = ["가", "나", "다", "라"]
  for 변수 in 리스트[1:]:
  print(변수)
  ```
  
  - 2자리를 증가해서 건너뜀
  ```
  리스트 = ["가","나","다","라"]
  for 변수 in 리스트[::2]:
  print(변수) 
  ```
  
  - 리스트를 거꾸로 나오게 출력
  ```
  리스트 = ["가","나","다","라"]
  for 변수 in 리스트[::-1]:
  print(변수)
  ```
